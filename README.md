**Arduino**와 **Processing**을 연동하여 식물의 생장 환경(습도, 온도, 조도)을 실시간으로 모니터링하고 시각화하는 프로젝트입니다.

> (여기에 실제 작동 사진 또는 Processing 화면 캡처를 넣어주세요)

---

## 📋 프로젝트 개요

식물을 키울 때 중요한 **습도(물)**, **조도(햇빛)**, **온도**를 센서로 측정합니다.  
측정된 데이터는:

- 아두이노가 **RGB LED와 부저**로 즉각적인 상태를 알려주고  
- USB 시리얼 통신을 통해 **PC의 Processing 대시보드에 실시간 그래프**로 표시됩니다.

---

## 🎯 주요 기능

### 🔍 실시간 모니터링
- **DHT11 센서**로 온도/습도 측정  
- **CdS(LDR)** 센서로 조도 측정  

### 🚨 상태 알림 (LED & 부저)
- 💧 **물 부족(건조)** → 빨간색 LED  
- ☀️ **햇빛 부족(어두움)** → 파란색 LED  
- 🌡️ **온도 경고** → 피에조 부저 경고음  
- 😊 **쾌적함(정상 상태)** → 초록색 LED  

### 🖥️ 데이터 시각화 (Processing)
- 아두이노가 전송한 데이터를 Processing이 수신  
- 그래프 + 실시간 수치로 대시보드 표시

## 🔌 회로 연결 (Pin Map)
# 🔌 스마트 반려식물 매니저 — 회로 연결도 (저항 제거 버전)

브레드보드를 사용하여 아래 순서대로 부품을 연결해 주세요.

---

## 1️⃣ 전원 연결 (기본)

- **Arduino 5V 핀 → 브레드보드 (+) 빨간 전원 라인**
- **Arduino GND 핀 → 브레드보드 (-) 파란 전원 라인**

---

## 2️⃣ DHT11 온습도 센서 (3핀 모듈 기준)

| DHT11 핀 | 연결 |
|---------|-------|
| VCC (+) | 브레드보드 5V 라인 |
| GND (-) | 브레드보드 GND 라인 |
| DATA (S) | Arduino **D2** |

---

## 3️⃣ RGB LED (공통 캐소드 타입)

| RGB LED 핀 | 연결 |
|------------|----------------------------|
| 공통 GND (가장 긴 다리) | 브레드보드 GND 라인 |
| 빨간색 R | Arduino **D4** |
| 초록색 G | Arduino **D5** |
| 파란색 B | Arduino **D6** |

---

## 4️⃣ 피에조 부저 (Piezo Buzzer)

| 부저 핀 | 연결 |
|---------|---------|
| (+) 긴 다리 | Arduino **D8** |
| (-) 짧은 다리 | 브레드보드 GND 라인 |

---

## 5️⃣ CdS 조도 센서 (LDR)

1. **CdS 센서 한쪽 다리 → 브레드보드 5V 라인**
2. **CdS 센서 다른 쪽 다리 → Arduino A0 핀**

---

## 📌 요약 회로 흐름 (저항 없음)

- CdS → A0  
- RGB LED → D4(DR), D5(DG), D6(DB)  
- 부저 → D8(+) → GND  
- DHT11 → D2

## 💻 소프트웨어 설정 (Software)

### 1️⃣ Arduino IDE 설정

#### 📦 라이브러리 설치
`Sketch → Include Library → Manage Libraries` 이동 후  
**"DHT sensor library by Adafruit"** 검색 → 설치

#### 📤 코드 업로드
`arduino/Smart_Plant_Sender.ino` 파일을 열고 업로드합니다.  
코드는 센서 값을 읽어 **습도,온도,조도** 형태의 CSV 데이터를 시리얼로 전송합니다.

---

### 2️⃣ Processing 설정

#### 💾 설치
Processing 공식 홈페이지 → 다운로드 & 설치

#### ▶️ 실행
`processing/Plant_Monitor_Dashboard.pde` 파일을 열고 실행(Run)합니다.  
아두이노가 연결된 상태에서 그래프가 실시간으로 갱신됩니다.

> ⚠️ 포트 번호 오류가 나면 `Serial.list()[0]` 인덱스를 조정하세요.

---

## 🚀 동작 로직 (Threshold)

| 상태 | 기준 값 | 동작 |
|------|--------|-------|
| **건조함** | 습도 < 40% | 🔴 빨간 LED |
| **어두움** | 조도 < 300 | 🔵 파란 LED |
| **온도 경고** | 온도 ≥ 30°C 또는 < 18°C | 🔊 부저 경고음 |
| **정상 상태** | 모든 기준 만족 | 🟢 초록 LED |
